## How to build

In the root directory, run the commands:
- (Linux & Windows) `cmake -S . -B build/`
- (Linux)           `make -C build/`

## How to run

### Linux

run `build/NEAT_C` in the command line.

### Windows

run the file `build_windows\Debug\NEAT_C.exe` after building the `NEAT_C` project in Visual Studio.
